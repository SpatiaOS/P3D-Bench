# Vendored Three.js (r183 / v0.183.2)

Only the files needed by the CAD benchmark are included:

- `build/three.core.js` — core library (ESM, self-contained)
- `build/three.module.js` — full library re-exporting core + WebGL renderer
- `examples/jsm/controls/OrbitControls.js` — orbit camera controls (HTML viewer)
- `examples/jsm/exporters/STLExporter.js` — STL export (Node.js headless pipeline)
